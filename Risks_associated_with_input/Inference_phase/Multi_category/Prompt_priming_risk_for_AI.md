# Prompt priming risk for AI

**Risks associated with input** \
**Inference** \
**Multi-category** \
**Amplified by generative AI**

### Description

Because generative models tend to produce output like the input provided, the model can be prompted to reveal specific kinds of information. For example, adding personal information in the prompt increases its likelihood of generating similar kinds of personal information in its output. If personal data was included as part of the modelâ€™s training, there is a possibility it could be revealed.

Depending on the content revealed, business entities could face fines, reputational harm, and other legal consequences.
